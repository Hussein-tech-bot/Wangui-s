# Hussein Wangui - Portfolio Website

A modern, responsive portfolio website showcasing my journey as a software developer. Built with vanilla HTML, CSS, and JavaScript, featuring interactive sections for projects and experiences.

![Portfolio Preview](./images/portfolio-preview.png)

## 🌟 Features

### ✨ Core Features
- **Responsive Design**: Fully responsive across all devices (mobile, tablet, desktop)
- **Interactive Projects Section**: Add, display, and manage portfolio projects
- **Dynamic Experiences Section**: Showcase work experience and career milestones
- **Smooth Navigation**: Single-page application with smooth scrolling
- **Modern UI/UX**: Clean design with gold accent colors and smooth animations
- **Data Persistence**: Projects and experiences saved in browser localStorage

### 🎯 Key Sections
- **About Me**: Personal introduction and background story
- **Skills**: Technical skills organized by categories (Frontend, Backend, UI/UX)
- **Projects**: Interactive project showcase with add/edit functionality
- **Experiences**: Professional experience timeline
- **Contact**: Social media links and contact information

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure

- **Font Awesome**: Icons for visual enhancement

### Development Tools
- **Git**: Version control

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic web server (optional for local development)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hussein-tech-bot/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for better development experience

3. **Using Python server (recommended)**
   ```bash
   # Python 3
   python3 -m http.server 8000

   # Or Python 2
   python -m SimpleHTTPServer 8000
   ```
   Then visit `http://localhost:8000`

## 💻 Usage

### Adding Projects
1. Click the **"+ Add Project"** button in the Projects section
2. Fill in the project details:
   - Project Title
   - Description
   - Technologies Used
   - Project Link (optional)
3. Click **"Save Project"** to add it to your portfolio

### Adding Experiences
1. Click the **"+ Add Experience"** button in the Experiences section
2. Fill in your experience details:
   - Job Title
   - Company Name
   - Duration (e.g., "Jan 2023 - Dec 2023")
   - Description of responsibilities
3. Click **"Save Experience"** to add it to your portfolio

### Managing Content
- **Delete Items**: Click the "Delete" button on any project or experience card
- **Data Persistence**: All data is automatically saved to your browser's local storage
- **Data Export**: Content persists between browser sessions
